一个用php描述的设计模式仓库

1. 俗话说：好记性不如烂笔头，记录下来，加深自己的印象
2. 烂笔头不如理解，使用示例来充分理解

### 怎么用

### 用什么方式

1. 创建型模式

    * 单例模式
2. 结构性模式
    * 适配器模式
    * 桥接模式
3. 行为型模式
    * 责任链
    * 观察者模式
    * 策略
